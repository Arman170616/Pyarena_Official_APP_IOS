# App Screenshots

## How to Take Screenshots

### Using iOS Simulator
1. Run the app: `Cmd + R`
2. Navigate to the screen you want to capture
3. Take screenshot: `Cmd + S` (saves to Desktop)
4. Or use: `Cmd + Shift + 4` for custom selection

### Required Screenshots

Please add the following screenshots to this folder:

1. **home-dashboard.png** - Main dashboard view
2. **about-us.png** - About section
3. **programs.png** - Programs listing
4. **teachers.png** - Teachers portfolio
5. **activities.png** - Activities list
6. **careers.png** - Career opportunities
7. **news.png** - News articles
8. **schools.png** - Schools directory
9. **contact.png** - Contact form
10. **chatbot.png** - AI chatbot interface
11. **admin-login.png** - Admin login screen
12. **more.png** - More options menu
13. **tab-navigation.png** - Bottom tab navigation

### Screenshot Guidelines

- **Device**: Use iPhone 15 Pro or iPhone 17 simulator
- **Orientation**: Portrait mode
- **Size**: Original simulator size (no need to resize)
- **Format**: PNG (lossless)
- **Content**: Make sure sample data is visible
- **Dark Mode**: Optional - can add both light and dark variants

### Naming Convention
Use kebab-case with descriptive names:
- ✅ home-dashboard.png
- ✅ teacher-detail-view.png
- ❌ Screen Shot 2025-12-03.png
- ❌ IMG_001.png

### Adding to Git
After adding screenshots:
```bash
git add screenshots/
git commit -m "Add app screenshots"
git push origin main
```
